<h1 align="center"> Conversor de moedas </h1>
<h2> Sobre: </h2>
<h3> Este é um desafio proposto pela Alura em parceria com a Oracle(ONE), O projeto consiste em uma aplicação em java para converter as moedas escolhidas pelo usuário. </h3>

https://github.com/user-attachments/assets/c064ea35-7857-4053-bfc0-4eb4572a25ed


:hammer: Funcionalidades do projeto:
<br>
- Assim que for baixado o arquivo adicione a depêndencia JAR.
- Configure a chave da API_KEY_CONVERSOR em uma Variável de Ambiente.
- O objetivo do desafio era criar uma aplicação capaz de consumir a API pública ExchangeRate-API, realizando requisições específicas e manipulando as respostas JSON para apresentar resultados de forma clara e organizada.

## ✔️ Tecnologias utilizadas

- ``Java``
- ``InteliJ IDEA``
- ``biblioteca Gson``
- ``Exchange Rate API``
