## :hammer: Como executar:
### Pré-requisitos:

- Java JDK 17 ou superior instalado
- Apache Maven instalado
- Certifique-se de ter o arquivo Gson em seu computador e adicione a dependência Jar ao projeto
- Configurar a API Key á variável de ambiente(API_KEY_CONVERSOR).

## ✔️ Tecnologias utilizadas

- ``Java JDK 17:`` Linguagem de programação
- ``InteliJ IDEA:`` Ambiente de Desenvolvimento Integrado (IDE)
- ``Gson 2.13.2:`` Biblioteca para serialização/desserialização JSON
- ``HttpClient:`` Módulo nativo do Java 11+ para requisições HTTP
- ``ExchangeRate-API:`` API principal para taxas de câmbio fornecendo informações atualizadas para conversões de moeda.``
